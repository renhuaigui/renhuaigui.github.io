搭建Hexo+Github的个人博客

配置

Node.js Git npm 必须安装 参见

Please tell me who you are 错误：

要配置自己得git账号：
git config –global user.name “github账号”
git config –global user.email “github账号邮箱”

注意：

在github上创建得repositories项目名字一定要为 自己的用户名.github.io
提交时如果不成功，可以删除目录下的 .deploy_git 尝试一下
